# Velkommen til Portal Nord 🧭

Dette er det første innlegget på **Portal Nord**, et nettsted bygget med [Hugo](https://gohugo.io/) og [PaperMod](https://github.com/adityatelange/hugo-PaperMod)-temaet.

## Hva er dette?

Portal Nord er en Hugo-drevet nettside som er klar til å fylles med innhold. Hugo er en av verdens raskeste rammeverk for statiske nettsteder.

## Kom i gang

Du kan opprette nye innlegg med:

```bash
hugo new content posts/mitt-innlegg.md
```

Og kjøre en lokal utviklingsserver med:

```bash
hugo server -D
```

God blogging! 🚀
